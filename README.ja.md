# blog

このリポジトリには、福井県越前市のプロジェクトである「国高キッズセーフマップ」に関するブログのコードが含まれています。

## デモ
[国高キッズセーフマップ](https://code4fukui.github.io/kunitaka/kidssafe.html)

## 機能
- CSVファイルによるデータ管理で更新が容易
- ドローン映像を活用し、通学路をVRで閲覧可能
- 街の情報などの機能にオープンデータを活用
- 地図上に不審者情報を追加

## データ / API
本プロジェクトでは、一部の地図機能にオープンデータを利用しています。
