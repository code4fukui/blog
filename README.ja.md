# blog

このリポジトリは、越前市の「国高地区 地域安全 キッズセーフマップ」に関する情報を管理しているブログです。

## デモ
- [国高地区 地域安全 キッズセーフマップ](https://code4fukui.github.io/kunitaka/kidssafe.html)

## 機能
- 児童の通学路をマップ上に表示
- 街頭情報をオープンデータから取得し表示
- 不審者情報をマップに追加

## データ・API
- オープンデータを活用したマップ作成
