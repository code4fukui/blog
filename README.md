# blog

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

This repository contains the code for a blog focused on the Kunitaka Kids Safe Map, a project in Echizen City, Fukui Prefecture, Japan.

## Demo
[Kunitaka Kids Safe Map](https://code4fukui.github.io/kunitaka/kidssafe.html)

## Features
- Manages data using CSV files for easy updates
- Allows viewing the school commute route in VR using drone footage
- Utilizes open data for features like street information
- Adds suspicious person information to the map

## Data / API
The project utilizes open data sources for some map features.
